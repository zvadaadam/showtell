# @showtell/compose

Renderer-owned presentation chrome shared by browser HyperFrames and timed media:

- captions and caption safe areas;
- presenter avatar/model badge overlays;
- watermark overlays;
- deterministic frame decoding/compositing;
- aspect-ratio dimensions, fonts, and resolved theme adapters.

Designed visuals do not live here. Browser HyperFrames are Showtell's only
visual design runtime; screencaps are timed media.
